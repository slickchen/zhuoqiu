# 玩家对战记录系统

一个基于Web的玩家对战记录系统，支持1v1和2v2对战模式，管理10位玩家（7位固定+3位自定义），记录战败方及备注信息。

## 功能特点

- **对战设置**：支持1v1和2v2两种对战模式
- **玩家管理**：7位固定玩家 + 3位自定义玩家
- **历史记录**：完整保存所有对战记录
- **数据导出**：支持将记录保存为图片
- **数据持久化**：使用浏览器本地存储保存数据
- **响应式设计**：适配手机和电脑浏览器

## 固定玩家

1. 第一陈
2. 斯提芬饭
3. 低质量黎
4. 复愁满伍
5. 鬼仔关
6. 十二生肖
7. 老王子米高

## 使用方法

1. 选择对战模式（1v1或2v2）
2. 为A队和B队选择玩家
3. 选择战败方（A队或B队）
4. 输入战败备注（可选，仅支持大写字母和数字）
5. 点击"记录对战结果"保存记录
6. 在"玩家管理"中可修改3位自定义玩家名称

## 技术栈

- HTML5
- CSS3 (Grid布局，Flexbox)
- JavaScript (ES6+)
- LocalStorage (数据存储)
- html2canvas (图片生成)
